# Vibe Report - EcoTrack

## 1) Como configure las reglas de mi agente
Configure el archivo `.cursorrules` para orientar al agente hacia un estilo de trabajo alineado con un MVP funcional y buenas practicas de desarrollo. Defini reglas claras para priorizar codigo limpio, modular y bien documentado, el uso de frameworks modernos (Next.js, React o Python con Streamlit), interfaces simples y enfoque en lo esencial del producto. Tambien incorpore lineamientos para que las correcciones de errores fueran claras, el manejo de API keys se hiciera con variables de entorno y cada generacion de codigo incluyera una breve explicacion.

Estas reglas funcionaron como una guia permanente durante el proyecto y ayudaron a mantener consistencia tecnica en cada iteracion.

## 2) Dificultades al delegar el codigo a la IA
La principal dificultad fue aprender a dar instrucciones precisas. Cuando el prompt era ambiguo, la salida no siempre respondia exactamente a lo que se buscaba y era necesario iterar. En la etapa de correccion de errores, la calidad de las respuestas mejoro cuando comparti el traceback completo y el contexto del archivo involucrado.

Otra dificultad fue cambiar el habito de corregir manualmente de inmediato. Delegar implica confiar en el flujo: describir bien el problema, evaluar la propuesta y ajustar la direccion en lugar de editar linea por linea.

## 3) Como se siente pasar de "escribir codigo" a "orquestar una vision"
El cambio se siente como pasar de una ejecucion tecnica detallada a una direccion de producto asistida por IA. En lugar de concentrarme en cada implementacion puntual, me enfoque en definir intencion, restricciones y criterios de calidad.

Este enfoque acelera la construccion de prototipos y permite validar ideas mas rapido. Al mismo tiempo, exige mayor claridad para comunicar objetivos y criterio para revisar resultados. La experiencia general fue positiva: mas velocidad para crear y mejor foco en la vision del producto.

## 4) Resultado
Se logro construir y desplegar un MVP funcional de EcoTrack, con una interfaz simple para registrar actividades diarias en lenguaje natural y estimar huella de CO2 de manera rapida, siguiendo una dinamica de iteracion continua con IA.
